# AI Workout Form Corrector

Real-time AI-powered workout form correction using computer vision and pose estimation.

## Features
- Real-time pose detection using MediaPipe
- Squat and push-up form analysis
- Live feedback and rep counting
- WebRTC camera integration

## Deployment
This app is deployed on Streamlit Cloud.

## Local Development
```bash
pip install -r requirements.txt
streamlit run app.py
```
